Presentation:
C'est un petit jeu type RogueLike, dans lequel vous contrôler un personnage qui doit avancer de salle en salle jusqu'a la salle "FIN".
Le personnage doit récupérer la clé de la salle pour ouvrir la porte et avancer dans le jeu.

Fonctionnalité Obligatoire :
    - Toutes les fonctions demandées obligatoires sont opérationnelles et respectent le modèle MVC.
    - La touche "c" permet d'utiliser les capsules lorsque le personnage est face au feu.
    - les objets sont ramassés automatiquement lorsque l'on marche sur celle-ci.
    - Les coffres peuvent donner une clé, une capsule ou un autre coffre (dans ce cas le second coffre est ouvert automatiquement).

Extension :
Ajouter le saut:
    - Lorsque le joueur est orienté en direction du trou et qu’une dalle non enflammée est accessible immédiatement de l’autre côté.
      Le saut permet de franchir le trou.

Scrolling :
    - On n'affiche que la salle dans  laquelle le joueur se trouve et on charge la bonne salle lorsque le joueur emprunt une porte.

Pour executer:
    - Aller dans le fichier bin puis lancer en ligne de commande "java -jar RogueLike.jar" pour lancer l'executable .jar. (Créer automatiquement avec VS Code)
    - OU Importer le projet sur un IDE type Intellij ou VisualStudio et exécuter le projet depuis le main.
